# Mahashe Install Hub

**Mahashe Install Hub** — приложение для **быстрой установки часто используемых программ на Windows** после переустановки системы.  
Вместо того чтобы вручную искать “миллион сайтов”, скачивать установщики и проверять версии — ты выбираешь нужные приложения в одном месте и устанавливаешь их быстрее и удобнее.

---

## Зачем это нужно

После чистой установки Windows обычно приходится заново ставить:
- браузер, архиватор, мессенджеры
- драйвер/утилиты, системные инструменты
- кодеки, редакторы
- рабочие приложения и т.п.

**Mahashe Install Hub** собирает это в одном интерфейсе, чтобы восстановление системы занимало минимум времени.

---

## Возможности

- Установка популярных приложений из одного окна
- Категории (например: Программы, Лаунчеры, Утилиты, Компоненты и т.д.)
- Пакетная установка (установить сразу несколько)
- Минималистичный интерфейс в стиле **Mahashe Install HUB**
- Экономит время после переустановки Windows
---

## Как это работает

1. Открываешь **Mahashe Install Hub**
2. Выбираешь нужные приложения
3. Нажимаешь установку — и не тратишь время на ручной поиск сайтов и установщиков

---

## Требования

- Windows 10 / Windows 11
- Права на установку приложений (обычно нужны админ-права)

---
